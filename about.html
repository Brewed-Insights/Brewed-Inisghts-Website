About Me

What I Do

Data Analytics & Visualization

Turning Data into Insights

I develop comprehensive Power BI solutions that transform complex datasets into clear, actionable insights. My work includes creating interactive dashboards, building advanced DAX measures, and designing semantic models that serve as the foundation for enterprise reporting.

  • Design and implement Power BI semantic models with complex relationships
  • Create dynamic reports using field parameters and advanced DAX calculations
  • Optimize query performance and model efficiency

Data Pipeline Development

Building Robust Data Infrastructure

I architect and maintain data pipelines using Microsoft Fabric and SQL Server, implementing best practices for data transformation and storage. My work ensures data flows smoothly from source systems to analytical platforms.

  • Develop medallion architecture patterns in Microsoft Fabric Lakehouses
  • Create efficient T-SQL procedures and transformations
  • Implement data quality checks and lineage tracking

Hospitality Analytics

Specialized Industry Solutions

I bring deep expertise in hospitality data analytics, working with hotel performance metrics, STR data, and management company transitions. My solutions help stakeholders understand property performance and make data-driven decisions.

  • Build custom analytics for hotel revenue management and performance tracking
  • Develop algorithms for detecting ownership transitions and market changes
  • Create comparative analysis tools for property benchmarking
